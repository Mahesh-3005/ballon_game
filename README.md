# 🎈 Color Balloon Mini Game (Sprout Assignment)

## 📱 Overview

This is a mini interactive learning screen built for children aged 3–5.
The goal is simple: children tap the correct colored balloon based on the instruction.

The experience is designed to be fun, responsive, and engaging within a short interaction loop.

---

## 🎯 Features

* Tap interaction (color-based selection)
* Animated floating balloons with wobble effect
* Pop (burst) animation with scale + fade
* Confetti particle effect on correct action
* Sound feedback for correct and incorrect taps
* Dynamic instructions (e.g., “Tap the RED balloon”)
* Score tracking system
* Kid-friendly UI (soft colors, rounded elements, playful layout)

---

## 🧠 Design Thinking

The interaction is designed around a simple loop:

Instruction → Action → Feedback → Reward → Repeat

Key focus areas:

* Immediate visual feedback
* Minimal cognitive load
* High responsiveness
* Engaging motion and sound

---

## 🎨 UI/UX Decisions

* Soft gradient background to mimic sky
* Floating balloon animations to create liveliness
* Rounded UI elements for child-friendly feel
* Non-blocking feedback placement for smooth gameplay

---

## ⚙️ Tech Stack

* Flutter
* GetX (state management)
* Audioplayers (sound effects)
* Confetti (particle animation)
